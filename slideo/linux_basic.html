Linux

Linux 是資安領域的基礎，但...什麼是Linux?

◈ 什麼是 Linux

      ▹ Linux是開源軟體發展中最著名的例子，
          也是單指作業系統的核心。
        而基於 Linux 的作業系統發行版則有需多，
          像是 Ubuntu, Kali 等等

◈ 登入 Kali 系統

      1.    打開KALI
      2.    帳號:kali 密碼:kali
      3.    搜尋http://23.146.248.199/challenges
      4.    註冊帳號
      5.    開啟命令提示字元

Kali 指令大全

記住這些指令，讓你的CTF技術更上一層樓

◈ >_ ls

      ▹ ls
          • 查看目錄
      ▹ ls -a
          • 列出所有檔案(包括隱藏檔案)
      ▹ ls -l
          • 列出文件詳細資訊

◈ >_ ..

      ▹ 回到上一層

◈ >_ cd

      ▹ 進入資料夾

◈ >_ cd ../

      ▹ 回到上一層資料夾
           (../回上一層,../..回到第二層,...以此類推)

◈ >_ pwd

      ▹ 查看位置

◈ >_ cat

      ▹ 查看內容

◈ >_ grep 關鍵字

      ▹ 在茫茫文字海當中找到關鍵字

◈ >_ grep -r 關鍵字 資料夾名稱

      ▹ 在多個資料夾中找到關鍵字位置
